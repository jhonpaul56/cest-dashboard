  1. Walk to the cabinet

### **Now (The New Way - This System)** 💻

  3. Boss is happy! 😊

### 3. **Add New Projects (Data Entry)** ✏️
**Like filling out a form, but the computer helps you!**

When DOST starts a new project:

   - Money: "₱150,000"
   - Year: "2024"
3. Click "Save"
4. Done! Computer remembers everything! 🎉

**The computer even warns you:**
- "Hey! This project already exists!" (prevents duplicates)
- "Hey! This budget seems too high!" (catches mistakes)
